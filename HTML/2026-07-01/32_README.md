# 개발자도구 (F12) — 화면 뒤를 들여다보는 꿀팁

## 개발자도구란?

브라우저에 숨어 있는 개발자도구로 화면 뒤의 **HTML·CSS가 다 보이고**, 값을 잠깐 바꿔 미리보기도 됨  
에러를 잡는 실전 도구입니다.

> ✅ 별도 설치 불필요 | 크롬·엣지 기본 내장

---

## 여는 법

- **Windows/Linux**: `F12` 키 누르기
- **Mac**: `Cmd + Opt + I`
- **또는** 화면 아무 곳 우클릭 → `검사`

---

## 🎯 꿀팁 — 이렇게 써먹기

### 1️⃣ 검사 모드로 요소 파악하기
화살표(검사) 아이콘 클릭 → 화면 요소에 마우스 올리면 그게 무슨 태그인지 콕 짚어 줌

### 2️⃣ Elements 탭으로 구조 확인하기
**Elements 탭** = 페이지 HTML 구조  
줄을 클릭하면 화면에서 그 부분이 반짝 표시

### 3️⃣ 디자인 미리보기
글자·색 값을 더블클릭해 바꿔보기 → 화면에 바로 반영

### 4️⃣ Console 탭에서 에러 찾기
**빨간 글씨** = 에러 메시지 → 어디가 틀렸는지 힌트

---

## ⚠️ 주의사항

> 여기서 바꾼 건 **화면에서만 잠깐** 표시됩니다
> - 내 파일은 안 바뀝니다
> - 새로고침하면 원래대로 돌아옵니다
> - 마음껏 실험해도 안전합니다 ✅
