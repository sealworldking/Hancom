# 1. 목적 — 왜 만들었는가

- 배열, 스택, 큐, 연결리스트 같은 자료구조는 글로만 읽으면 추상적이라 입문자가 이해하기 어렵다.
- '읽기'가 아니라 '직접 카드를 드래그해서 쌓고 빼보는' 경험을 주면 규칙이 몸에 남는다는 가정에서 출발했다.
- 기존 학습 자료 대부분은 텍스트 + 정적 다이어그램 위주라서, 설명 → 시연 → 직접 해보기 → 채점으로 이어지는 4단계 체험형 구조로 차별화했다.

# 2. 대상 이용자

- 코딩을 갓 시작해 배열까지는 알지만 스택, 큐, 연결리스트는 처음 듣는 입문자
- 전공 수업이나 코딩테스트 대비로 자료구조 개념을 짧게 복습하고 싶은 사람

# 3. 핵심 기능

# 3.1 시작 화면 (index.html)
- 마스코트 말풍선이 아직 안 배운 자료구조를 무작위 멘트로 추천 (클릭 시 다음 멘트로 순환)
- 자료구조별 카드 메뉴 4종(배열, 스택, 큐, 연결리스트) + 준비중 카드 2종(해시테이블, 이진트리)
- `localStorage`에 학습 완료 여부를 저장해, 완료한 카드에는 완료 뱃지를 표시
- 하단 정보띠에 전체 진행률(예: 2/4) 표시

# 3.2 학습 페이지 공통 3-Step 구조
자료구조 4종(배열, 스택, 큐, 연결리스트) 모두 동일한 패턴으로 반복된다.

Step 1. 설명, 정의, 규칙, 비유를 카드로 제시
Step 2. 시연, '다음 연산' 버튼을 눌러 한 단계씩 직접 진행하며 push/pop, enqueue/dequeue 등 동작을 관찰 (처음부터 다시보기 가능)
Step 3. 직접 해보기 + 채점, 뒤섞인 카드를 드래그 앤 드롭으로 규칙에 맞게 배치 → 채점 → 맞은 개수/점수/등급(완벽, 잘함, 다시 도전) 표시, 재도전 가능

# 3.3 자료구조별 채점 규칙

배열: 숫자 카드를 오름차순으로 드롭,  배치 순서 == 오름차순 정렬 배열
스택: 지정된 push 순서대로 드롭(맨 위로 쌓임), 드롭 순서 == 지정 push 순서
큐: 지정된 enqueue 순서대로 드롭(맨 뒤로 붙음), 드롭 순서 == 지정 enqueue 순서
연결리스트: 지정된 순서대로 체인 끝에 연결 + 삭제 연습(이전→다음 노드 순서 클릭), 연결 순서 == 정답 순서

- 점수: `(정답 위치 일치 개수 / 전체) * 100`
- 등급: 90점 이상 '완벽', 70점 이상 '잘함', 미만 '다시 도전'

# 3.4 그 외 디테일
- 클릭할 때마다 잉크가 번지는 듯한 물결 효과(reduced-motion 설정 시 비활성화)
- 배경에 항상 일렁이는 파도 애니메이션(Canvas)으로 '자료구조 놀이터'라는 톤 유지

# 4. 화면 구성

상단바(topbar): 메뉴로 돌아가기 링크, Step 진행 표시 점(dot)
Header: 자료구조 이름 + 마스코트 말풍선
Step 패널: 설명 카드 / 시연 스테이지 / 드래그 풀, 드롭존, 채점 결과가 Step별로 전환되며 노출
하단 정보띠(footer): 제작자, 토픽 번호, 업데이트일, 전체 진행률

# 5. 정보 구조 (사이트맵)

```
index.html            (시작 화면 — 커리큘럼 메뉴 + 진행률)
├─ array.html          (배열, 정렬)
├─ stack.html          (스택)
├─ queue.html          (큐)
└─ linkedlist.html     (연결리스트 — 삽입 + 삭제 연습 포함)
```
